Retrieve youtube videos from a user display those as the video thumbnail and open those in a modal.

== Description ==

Youtube video User is a plugin that display a widget in which you add a user of youtube and it will retrieve the defined number of videos and display the thumbnails of those in an unordered list, once you click on the thumbnail, the video will show inside a prettyphoto modal box.

Thuis plugin is ready for translation, and comes with the french version of it.

== Installation ==

1. Upload to the `/wp-content/plugins/` directory
1. Activate the plugin through the "Plugins" menu in WordPress
1. Add the widget to one of your widgetized area
1. See the result

== Frequently Asked Questions ==

= What about if i want to use a user from vimeo =

For now this plugin just works with youtube, but i will try to add vimeo and dailymotin for the next update

= How can i change the prettyphoto settings =

For now no options are configurable, but soon will come the full prettyphoto options for you to choose from.
